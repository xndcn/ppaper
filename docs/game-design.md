## 一、游戏概念

### 电梯简介

**折翼传说（PPaper）** 是一款以纸飞机为核心的 roguelike 2D 像素风浏览器游戏。玩家扮演一个热爱折纸飞机的少年，从校园小赛场出发，收集并折叠各种纸飞机，挑战一个又一个对手，最终登上世界纸飞机大赛的舞台。

### 定位

| 维度 | 设定 |
|------|------|
| 目标受众 | 休闲玩家、学生群体，8 岁以上 |
| 单局时长 | 5-15 分钟（适合碎片时间） |
| 核心乐趣 | 物理感 × 收集成就感 × 每局不重复的 Roguelike 惊喜 |
| 灵感来源 | Pokemon（收集/对战进阶）、Angry Birds（物理投掷）、Slay the Spire（Roguelike 循环）、动画《纸飞机》 |

### 游戏愿景

> 让每一架纸飞机都有自己的故事——不同的折法、不同的属性、不同的命运。

## 二、核心游戏循环

### 宏观循环（跨局）

```
[开始游戏]
    ↓
[探索大世界] ← 永久解锁的城镇地图
    ↓
[折纸工坊] — 选择/改进纸飞机
    ↓
[进入比赛] — 消耗"比赛入场券"
    ↓
[比赛循环] ← Roguelike 核心
    ├── 回合 1：随机赛道 + 掉落奖励
    ├── 回合 2：难度递增
    └── ... 失败则本局结束
    ↓
[结算] — 金折（永久货币）+ 图鉴记录
    ↓
[回到大世界] — 解锁新内容，准备下一局
```

### 微观循环（单次比赛）

```
[加载赛道] — 程序化生成风向/障碍/目标区域
    ↓
[发射阶段] — 拖动调整角度 + 力度（类似愤怒的小鸟）
    ↓
[飞行阶段] — 实时物理（升力/阻力/重力/风）+ 可选技能
    ↓
[落地/计分] — 距离 × 精准度 × 特技加成
    ↓
[对比对手] — 胜/负/平
    ↓
[获得奖励] — 技能/物品/纸飞机碎片
```

## 三、场景设计

### 3.1 主菜单（MainMenuScene）

- 背景：像素风校园操场
- 元素：飞翔的纸飞机动画（粒子系统）
- 按钮：新游戏、继续游戏、图鉴、设置
- 音乐：轻快的芯片音乐

### 3.2 大世界地图（OverworldScene）

俯视角像素城镇，玩家可自由行走，与 NPC 互动。

| 地点 | 功能 |
|------|------|
| 🏠 家（Home） | 存档、查看进度、获取每日任务 |
| 🏫 学校（School） | 触发主线剧情、与同学对战 |
| ⛰️ 后山（Back Mountain） | 特殊事件、隐藏纸飞机图纸 |
| 🏪 小卖部（Shop） | 购买物品、消耗品、普通图纸 |
| 🏆 竞技场（Arena） | 进入各级比赛 |
| 🌳 公园（Park） | 自由练习场，体验不同纸飞机 |
| 🔧 工坊（Workshop） | 折纸工坊入口 |

**地图规格：**
- 尺寸：80×60 方块（每方块 32×32 像素）
- 视角：略微倾斜的俯视（类 Pokemon）
- 支持 NPC 随机漫步，节日活动更换装饰

### 3.3 折纸工坊（WorkshopScene）

玩家可视化地调整纸飞机折叠参数：

```
┌─────────────────────────────────────────┐
│   当前纸飞机：标准飞镖 ★★☆☆☆           │
│                                         │
│    /\                                   │
│   /  \    ←  实时飞行特性预览            │
│  /____\                                 │
│                                         │
│  鼻锥重量 ████░░  [←→]   影响：俯冲速度  │
│  翼展角度 ██░░░░  [←→]   影响：滑翔距离  │
│  尾翼角度 ███░░░  [←→]   影响：稳定性    │
│                                         │
│  [确认折叠]  [重置]  [查看折法步骤]       │

```

- 每次调整参数后，右侧实时显示预期飞行曲线
- 「查看折法步骤」展示真实纸飞机折叠教程（ASCII 图解 + 文字说明）
- 折叠操作消耗「纸张」资源

### 3.4 比赛场景（CompetitionScene）

横版侧视视角，玩家与若干 AI 对手竞争。

**赛道布局：**
```
[起点]  ←←←  风向可视化（粒子/箭头）  →→→  [终点目标区]
  🏢                                              🎯
   ↑                                              ↑
  发射台                          ★精准落点★（额外分）
        🪨 障碍物（随机）   🌊 上升气流区（随机）
```

**发射交互（移动端优先）：**
```
手机：拖动飞机调整角度 → 松手发射
桌面：鼠标拖动 → 松开发射，或方向键+空格
```

**飞行阶段 UI：**
- 屏幕角落显示当前风向/风速
- 玩家飞机可在飞行中轻微调整俯仰（滑动/方向键）
- 技能按钮（主动技能最多 3 个）

**对手视觉：**
- 其他纸飞机同屏显示，颜色区分
- 落点记录在赛道末端

## 四、纸飞机系统

### 4.1 四大基础类型

| 类型 | 原型 | 速度 | 稳定性 | 升力 | 特点 | 适合场景 |
|------|------|------|--------|------|------|----------|
| 飞镖型（Dart） | 标准飞镖 | ⚡⚡⚡⚡ | ⭐⭐ | ⭐ | 直线穿透力强，抗侧风差 | 顺风、无障碍赛道 |
| 滑翔型（Glider） | 宽翼滑翔机 | ⚡⚡ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 飞行时间长，距离远 | 追求最远距离 |
| 特技型（Acrobatic） | 燕子型 | ⚡⚡⚡ | ⭐⭐⭐ | ⭐⭐⭐ | 可执行翻滚/环绕，特技加分 | 特技赛/障碍赛 |
| 花式型（Stunt） | 不规则折叠 | ⚡⚡⚡ | ⭐ | ⭐⭐ | 独特飞行轨迹，特殊技能 | 特定赛事 |

### 4.2 六维属性

```
速度（Speed）：初始投掷速度上限
稳定性（Stability）：受侧风/乱流影响的抵抗力
升力（Lift）：在相同速度下产生的升力大小
耐久（Durability）：与障碍物碰撞后的耐久损耗
重量（Weight）：影响重力加速下降的速率
幸运（Luck）：比赛后掉落稀有物品的概率加成
```

属性范围：1-10，每项最高 10 分。

### 4.3 折叠参数（影响飞行物理）

| 参数 | 范围 | 物理影响 |
|------|------|----------|
| 鼻锥重量（Nose Weight） | 0-5 | 重心前移 → 稳定性↑、速度↓ |
| 翼展（Wingspan） | 0-5 | 翼面积↑ → 升力↑、阻力↑ |
| 翼弦角（Dihedral Angle） | -3~+3 | 正值→抬升稳定，负值→急速俯冲 |
| 尾翼角度（Tail Angle） | -3~+3 | 影响飞行路径曲率 |

### 4.4 稀有度与图纸系统

| 稀有度 | 颜色 | 获取方式 | 属性上限 | LLM 生成名称 |
|--------|------|----------|----------|-------------|
| 普通（Common） | 灰 | 商店购买 | 总分 ≤ 20 | ❌ 预设名称 |
| 稀有（Rare） | 蓝 | 比赛掉落 | 总分 ≤ 30 | ✅ |
| 精良（Epic） | 紫 | Boss 奖励 | 总分 ≤ 40 | ✅ |
| 传说（Legendary） | 金 | 特殊事件 | 总分 ≤ 50 | ✅（详细背景故事）|

**图纸碎片机制：** 收集同款图纸碎片可升级纸飞机（类 Pokemon IV 系统），重复的传说图纸碎片特别珍贵。

## 五、物理模型

### 5.1 飞行力学（二维简化模型）

每帧更新（以固定时间步长 1/60 秒）：

```
1. 计算当前攻角 α = pitch_angle - velocity_angle
2. 计算升力系数 C_l：
   - α < 15°：C_l = 2π × α（线性区）
   - 15° ≤ α < 25°：C_l 线性下降（失速过渡）
   - α ≥ 25°：C_l = 0（完全失速）
3. 计算阻力系数 C_d：
   C_d = C_d0 + C_l² / (π × AR × e)
   其中 AR=翼展比，e=奥斯瓦尔德效率因子 ≈ 0.8
4. 计算各力向量：
   - 重力 F_g = (0, -m×g)
   - 升力 F_l = C_l × 0.5 × ρ × v² × A × 法向向量
   - 阻力 F_d = -C_d × 0.5 × ρ × v² × A × 速度方向单位向量
5. 合力 → 加速度 → 更新速度/位置
```

### 5.2 风力系统

```
赛道风力 = 基础风（恒定） + 阵风（随机脉冲） + 热气流（局部区域）

基础风：  方向和强度在赛道生成时随机确定，全局恒定
阵风：    泊松分布触发，持续 0.5-2 秒，随机方向偏移 ±45°
热气流：  赛道特定区域存在向上升力（可视化为涡旋图案）
```

**风力视觉反馈：**
- 背景层粒子（落叶、尘埃）展示风向
- 飞机飞入侧风时机身轻微倾斜

### 5.3 不同类型纸飞机的物理参数

| 类型 | 质量 m | 翼面积 A | 翼展比 AR | C_d0 | 初始速度范围 |
|------|--------|----------|-----------|------|-------------|
| 飞镖型 | 0.012 kg | 0.01 m² | 3 | 0.05 | 8-15 m/s |
| 滑翔型 | 0.008 kg | 0.02 m² | 6 | 0.03 | 5-10 m/s |
| 特技型 | 0.010 kg | 0.015 m² | 4 | 0.04 | 7-12 m/s |
| 花式型 | 0.010 kg | 0.012 m² | 3.5 | 0.06 | 6-12 m/s |

### 5.4 攻角控制

飞行中玩家可有限调整俯仰：
- 移动端：上下滑动
- 桌面：方向键上/下
- 调整灵敏度受「稳定性」属性影响（高稳定性 → 调整更缓慢但更精准）

## 六、Roguelike 要素

### 6.1 Roguelike 运行结构

```
一次"挑战赛"（Run）结构：
  选择纸飞机 + 初始技能（2选1）
    ↓
  第 1 回合：入门级赛道
    ↓（胜利 → 随机选一项奖励）
  第 2 回合：难度↑，新风向/障碍
    ↓（胜利 → 奖励）
  ...
  第 5 回合：BOSS 关
    ↓（胜利 → 完成本赛事，获得大量金折）
    （失败 → 本局结束，本局纸张/技能清空，金折保留）
```

### 6.2 赛道程序化生成（每回合）

通过 rot.js RNG 生成赛道参数：

| 参数 | 生成逻辑 |
|------|----------|
| 赛道长度 | 基础 30m，每回合 +5m |
| 风向 | 随机 8 方向，强度 0-8 m/s |
| 障碍物数量 | 0-3 个（按难度递增） |
| 障碍物类型 | 树、建筑、气球（可击破获分） |
| 热气流区域 | 0-2 个，随机位置 |
| 精准落点位置 | 随机偏移目标区 |
| 对手数量 | 1-4 个 |

**种子系统：** 每局 Run 有一个可分享的 6 位种子码，好友可用相同种子体验相同赛道序列。

### 6.3 随机奖励（每回合结算后 3选1）

```
类型 A：技能
  - 随机 1 个主动技能或被动技能
  - 提示已有技能的协同效果

类型 B：纸飞机升级
  - 当前飞机某属性 +1
  - 或获得一张随机图纸碎片

类型 C：物品
  - 1-2 个消耗品（下一回合使用）
  - 或永久微型装备
```

### 6.4 元进度（跨局永久保留）

| 永久保留 | 描述 |
|----------|------|
| 金折（Gold Folds） | 比赛奖励的货币，用于商店购买 |
| 图鉴解锁 | 首次获得的纸飞机/技能永久记录 |
| 折法教程 | 解锁后的真实折纸步骤永久保存 |
| 锦标赛进度 | 已解锁的赛事层级 |
| 成就 | 特殊条件解锁的徽章 |

### 6.5 难度设计

```
难度公式（每回合）：
  目标距离 × (1 + 0.15 × round)
  风力强度 × (1 + 0.10 × round)
  障碍物密度 × (1 + 0.20 × round)

对手 AI 水平：
  回合 1-2：随机发射（最弱）
  回合 3-4：使用简单技能
  回合 5（Boss）：使用完整技能组合
```

## 七、技能系统

### 7.1 主动技能（最多携带 3 个）

| 技能名称 | 效果 | 冷却 | 解锁条件 |
|----------|------|------|----------|
| 顺风加速 | 临时推力 +40%，持续 1.5 秒 | 1次/局 | 初始解锁 |
| 风盾 | 降低侧风影响 60%，持续 3 秒 | 1次/局 | 商店购买 |
| 俯冲 | 强制俯冲到最优攻角，瞬间触发 | 2次/局 | 比赛掉落 |
| 翻筋斗 | 完成一个 360° 翻转，获得特技加分 | 1次/局 | 赛事奖励 |
| 纸翼展开 | 临时提升翼面积 30%，增加升力 | 1次/局 | Boss 掉落 |
| 急速折叠 | 当前回合完成后可重新折叠纸飞机 | 1次/局 | 传说物品 |

### 7.2 被动技能（最多携带 3 个）

| 技能名称 | 效果 | 叠加规则 |
|----------|------|----------|
| 顺风亲和 | 顺风时升力 +20% | 可叠加 2 次 |
| 精工折叠 | 纸飞机所有属性 +5% | 不可叠加 |
| 纸张质量 | 耐久 +2，碰撞障碍不失速 | 不可叠加 |
| 幸运折痕 | 掉落稀有物品概率 +15% | 可叠加 2 次 |
| 轻量化 | 重量 -1，飞行时间 +10% | 不可叠加 |
| 气流感知 | 热气流区域在 UI 上标记 | 不可叠加 |

### 7.3 技能协同（部分组合有额外效果）

| 组合 | 协同效果 |
|------|----------|
| 顺风加速 + 顺风亲和 | 激活时距离 +20% 额外加成 |
| 翻筋斗 + 特技型纸飞机 | 翻筋斗得分加倍 |
| 纸张质量 + 轻量化 | 解锁"纸皮铠甲"效果：碰撞弹开而非扣耐久 |

## 九、对手系统

### 9.1 普通对手类型

| 对手名称 | 招牌纸飞机 | 难度 | 特点 |
|----------|------------|------|------|
| 小明（同班同学） | 标准飞镖 | ⭐ | 入门教学关 |

## 十三、LLM 动态内容

### 生成内容类型

| 内容 | 触发时机 | 模型大小 | JSON 输出格式 |
|------|----------|----------|---------------|
| 稀有/传说纸飞机名称 | 首次获得 | 1.5B | `{"name": "...", "subtitle": "..."}` |
| 纸飞机背景故事 | 传说级获得 | 3B | `{"story": "..."}` |
| 技能趣味文案 | 首次解锁 | 1.5B | `{"flavor_text": "..."}` |
| 赛前对手对话 | Boss 关 | 3B | `{"dialogue": "..."}` |
| 新折纸设计参数 | 特殊事件 | 7B | `{"type": "...", "attributes": {...}}` |

### 内容安全校验

所有 LLM 生成内容必须通过 `ContentValidator` 校验：
1. JSON Schema 格式校验（防止解析错误）
2. 内容长度限制（名称 ≤ 10 字，故事 ≤ 200 字）
3. 敏感词过滤
4. 属性数值范围校验（防止生成超出范围的数值）

校验失败则使用预设的静态兜底内容。

### 降级策略

```
LLM 可用（WebGPU 支持）→ 客户端推理生成
LLM 不可用               → 从 src/data/ 静态内容库随机选取
网络可用 + 用户配置 API Key → 服务端 API 生成（可选开启）
```
